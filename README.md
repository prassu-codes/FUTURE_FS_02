# NexusFlow – Relationship Intelligence CRM

🚀 **NexusFlow** is a modern **Client Lead Management System (Mini CRM)** developed as part of my **Full Stack Web Development Internship – Task 2 at Future Interns**.

The application helps businesses manage leads, track client relationships, and monitor deal progress through an intelligent pipeline system.

This project was built using **AI-assisted development with Lovable**, combined with custom modifications and testing.

---

## 📌 Project Objective

The goal of this project is to create a **functional CRM system** that helps manage client leads generated from various sources and track their progress through different stages of a sales pipeline.

---

## ✨ Features

✔ Lead Management System
✔ Client Information Storage (Name, Email, Company, Source)
✔ Lead Status Pipeline

```text
Prospect → Contacted → Negotiation → Won → Lost
```

✔ Notes and follow-up tracking
✔ Lead search and filtering
✔ Dashboard statistics for lead insights
✔ User-friendly interface for managing relationships

---

## 🧠 Unique Concept

Unlike traditional CRMs that use simple tables, **NexusFlow focuses on relationship intelligence** by visually organizing client interactions and pipeline progress, helping users understand the **health and status of each lead**.
